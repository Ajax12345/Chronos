Features

Create events

Select dates and then choose the event visibility, either public or private. Select user-added timeslots to finalize the event

Event timerange selection

Chronos enables event participants to select their own timeslots when they are available, on the days you choose.

Event messages

Easily communicate between all event members.

Event visibility

Events can be public or private. Private events are only accessible to users or groups added by you, while any user can add availability to a public event.

Groups

Groups can be created by adding users and other groups. Groups are portable, and can be added to any event.

Timerange preferences

Users can select their availablity preference for each timeslot. An algorithm utilizes these rankings to select timeslots consisting of most convient times for each event member.

Event settings

Finalize, cancel, or delete an event. Remove previously added users and groups, or add additional users and groups.

Automatic timeslot selection

Can't wait for users to add their own availability? Chronos can automatically find available timeslots for each event user, based on their personal calendar.

Timeslot overlap prevention

Timelistings display your previously-scheduled events, so you do not need to check back and forth between multiple calendars as you add your availability.
